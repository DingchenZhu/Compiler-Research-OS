# Skills

Personal skills for Compiler Research OS.

These are reusable operating procedures for Codex and other AI agents. Each skill should live in its own folder and include a required `SKILL.md`.

Recommended structure:

```text
13_skills/
├── README.md
└── skill-name/
    ├── SKILL.md
    └── references/
```

Principles:

- Keep `SKILL.md` concise.
- Put long workflows, schemas, and examples in `references/`.
- Prefer Markdown, YAML, and shell commands that work locally.
- Skills should describe how to operate the OS, not merely explain concepts.
